# find-skills

## 简介

搜索和发现适合当前任务的社区 Skill。

## 来源

[vercel-labs/skills](https://github.com/vercel-labs/skills)，路径 `skills/find-skills`。

## 安装方案

上游安装：`npx skills add vercel-labs/skills@find-skills -y -g`。

本仓库统一安装：在仓库根目录运行 `./scripts/install.sh`，脚本完成公共源码更新后，将该 Skill 复制到 `~/.agents/skills/find-skills`。
