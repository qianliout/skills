# skill-creator

## 简介

指导创建和改进包含说明、脚本、参考资料与资源的 Skill。

## 来源

[anthropics/skills](https://github.com/anthropics/skills)，路径 `skills/skill-creator`。

## 安装方案

上游安装：`npx skills add anthropics/skills@skill-creator -y -g`。

本仓库统一安装：在仓库根目录运行 `./scripts/install.sh`，脚本完成公共源码更新后，将该 Skill 复制到 `~/.agents/skills/skill-creator`。
