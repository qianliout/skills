# requesting-code-review

## 简介

在完成任务或准备合并前组织结构化代码审查。

## 来源

[obra/superpowers](https://github.com/obra/superpowers)，路径 `skills/requesting-code-review`。

## 安装方案

上游安装：`npx skills add obra/superpowers@requesting-code-review -y -g`。

本仓库统一安装：在仓库根目录运行 `./scripts/install.sh`，脚本完成公共源码更新后，将该 Skill 复制到 `~/.agents/skills/requesting-code-review`。
