# obsidian-markdown

## 简介

创建和编辑包含双链、嵌入、属性与 Callout 的 Obsidian Markdown。

## 来源

[kepano/obsidian-skills](https://github.com/kepano/obsidian-skills)，路径 `skills/obsidian-markdown`。

## 安装方案

上游安装：`npx skills add kepano/obsidian-skills@obsidian-markdown -y -g`。

本仓库统一安装：在仓库根目录运行 `./scripts/install.sh`，脚本完成公共源码更新后，将该 Skill 复制到 `~/.agents/skills/obsidian-markdown`。
