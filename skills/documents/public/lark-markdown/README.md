# lark-markdown

## 简介

查看、创建、上传、编辑和比较飞书 Markdown 文件。

## 来源

[larksuite/cli](https://github.com/larksuite/cli)，路径 `skills/lark-markdown`。

## 安装方案

上游安装：`npx skills add larksuite/cli@lark-markdown -y -g`。

本仓库统一安装：在仓库根目录运行 `./scripts/install.sh`，脚本完成公共源码更新后，将该 Skill 复制到 `~/.agents/skills/lark-markdown`。
