# vercel-react-best-practices

## 简介

按照 Vercel 工程实践优化 React 和 Next.js 的性能与加载行为。

## 来源

[vercel-labs/agent-skills](https://github.com/vercel-labs/agent-skills)，路径 `skills/react-best-practices`。

## 安装方案

上游安装：`npx skills add vercel-labs/agent-skills@vercel-react-best-practices -y -g`。

本仓库统一安装：在仓库根目录运行 `./scripts/install.sh`，脚本完成公共源码更新后，将该 Skill 复制到 `~/.agents/skills/vercel-react-best-practices`。
