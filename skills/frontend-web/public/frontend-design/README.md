# frontend-design

## 简介

设计并实现具有明确视觉风格的生产级前端页面和组件。

## 来源

[anthropics/skills](https://github.com/anthropics/skills)，路径 `skills/frontend-design`。

## 安装方案

上游安装：`npx skills add anthropics/skills@frontend-design -y -g`。

本仓库统一安装：在仓库根目录运行 `./scripts/install.sh`，脚本完成公共源码更新后，将该 Skill 复制到 `~/.agents/skills/frontend-design`。
