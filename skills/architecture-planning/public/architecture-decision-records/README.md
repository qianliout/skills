# architecture-decision-records

## 简介

编写和维护架构决策记录，保存重要技术决策的背景、选项和结果。

## 来源

[wshobson/agents](https://github.com/wshobson/agents)，路径 `plugins/documentation-generation/skills/architecture-decision-records`。

## 安装方案

上游安装：`npx skills add wshobson/agents@architecture-decision-records -y -g`。

本仓库统一安装：在仓库根目录运行 `./scripts/install.sh`，脚本完成公共源码更新后，将该 Skill 复制到 `~/.agents/skills/architecture-decision-records`。
