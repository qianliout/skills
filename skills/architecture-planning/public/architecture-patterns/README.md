# architecture-patterns

## 简介

指导 Clean Architecture、六边形架构和领域驱动设计等后端架构实践。

## 来源

[wshobson/agents](https://github.com/wshobson/agents)，路径 `plugins/backend-development/skills/architecture-patterns`。

## 安装方案

上游安装：`npx skills add wshobson/agents@architecture-patterns -y -g`。

本仓库统一安装：在仓库根目录运行 `./scripts/install.sh`，脚本完成公共源码更新后，将该 Skill 复制到 `~/.agents/skills/architecture-patterns`。
