# project-planner

## 简介

将复杂项目拆解成阶段、任务、依赖、里程碑和风险清单。

## 来源

[shubhamsaboo/awesome-llm-apps](https://github.com/shubhamsaboo/awesome-llm-apps)，路径 `awesome_agent_skills/project-planner`。

## 安装方案

上游安装：`npx skills add shubhamsaboo/awesome-llm-apps@project-planner -y -g`。

本仓库统一安装：在仓库根目录运行 `./scripts/install.sh`，脚本完成公共源码更新后，将该 Skill 复制到 `~/.agents/skills/project-planner`。
